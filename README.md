# Joseph Hilte Portfolio Website

A simple employer-facing portfolio website built with HTML, CSS, and JavaScript.

## Sections

- About
- Experience
- Projects
- Leadership
- Skills
- Contact

## How to Run Locally

1. Download or clone this repository.
2. Open the folder in VS Code.
3. Open `index.html` in your browser.

Optional: install the VS Code extension **Live Server**, right-click `index.html`, and choose **Open with Live Server**.

## How to Customize

- Replace the LinkedIn and GitHub placeholder links in `index.html`.
- Update project descriptions as Joseph adds GitHub repositories.
- Replace `Joseph_Hilte_Resume.pdf` with the newest resume when needed.

## Suggested GitHub Pages Setup

1. Create a new public repository on GitHub, such as `joseph-portfolio`.
2. Upload these files to the repository:
   - `index.html`
   - `style.css`
   - `script.js`
   - `Joseph_Hilte_Resume.pdf`
   - `README.md`
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save, then wait for GitHub Pages to publish the site.
